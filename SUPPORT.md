# Support

Maintainer Compass is maintained as a small open-source project. The goal is to
keep support lightweight, public, and useful for future contributors.

## Where to Ask

- Use GitHub issues for reproducible bugs, feature requests, and documentation
  gaps.
- Use pull requests for focused improvements with tests or documentation.
- Use GitHub security advisories, if available, for private vulnerability
  reports.

## What to Include

For bugs, include:

- The command you ran.
- The repository layout or minimal sample that reproduces the issue.
- The expected and actual report output.
- Your Python version and operating system.

For feature requests, include:

- The maintainer workflow you want to improve.
- The signal or report section you expect Maintainer Compass to produce.
- Any examples from real OSS maintenance work.

## Response Expectations

This project is maintained asynchronously. Issues that include a reproduction,
clear scope, and a suggested test case are easier to review quickly.
