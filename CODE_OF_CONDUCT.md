# Code of Conduct

This project follows the Contributor Covenant spirit: be respectful, direct,
and constructive.

Expected behavior:

- Use welcoming language.
- Respect different experience levels and backgrounds.
- Focus criticism on code and project outcomes.
- Assume good intent, but correct harmful behavior quickly.

Unacceptable behavior:

- Harassment, threats, or discriminatory language.
- Publishing private information without permission.
- Repeated disruption of project discussions.

Project maintainers may remove comments, close issues, or block contributors
who violate these expectations.
